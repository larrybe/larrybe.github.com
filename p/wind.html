An experiment combining WebGL, WebRTC and Web Audio API. I was inspired to make this after watching an author showcase his interactive story book on TED Talk.
NOTE: This project requires access to your microphone. ALSO: This was created a while ago and doesn't take advantage of any recent improvements in the Babylon.js engine. I plan on updating this project soon.

built with: babylon.js/WebGL, Web Audio API, WebRTC, 3ds Max

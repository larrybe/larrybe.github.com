paperplane

I wanted to test out Unity's WebGL capabilities so I decided to make a gliding game with it. I found that even though it is not perfect and has more than a few frustrating bugs, it is one of the most capable and feature rich WebGL engines available.

NOTE: This is unfinshed so there are certain broken aspects: controls are a bit unpredictable, collisions and physics are a bit erratic, etc. - bear with me.

built with: Unity/WebGL, C#
